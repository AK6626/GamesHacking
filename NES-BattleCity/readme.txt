Tank 1989 and Tank 1990 are two famous hacks by YanShan in China.
The Tank 1989 sign the game code on the map, which is annoying a lot.
The 'Remove ...' ips hack is to solve the problem. Of course you should apply the Tank 1989 hack first.

Use a tool like 'Lunar IPS' to apply the ips on the rom file.
